UX Researcher

User Experience

UX Researcher

UX 研究員主要透過各式各樣的研究方法，了解市場及使用者的需求，研究結果用以優化 WEB、APP 等各類產品服務。工作內容主要為研究問題擬定與實驗設計、研究執行、成果報告撰寫。舉凡文獻回顧、使用者測試、田野研究（民俗誌，行人觀察，情境訪談）、個人訪談與焦點團體、問卷調查與統計分析、使用者規範撰寫（UX Spec）等皆為 UX 研究員需熟悉的技能。 適合求知慾強、積極解決問題、對人類行為感到好奇，想要深入了解的你。

Web/App UI Designer

User Interface

Web/App UI Designer

UI設計師負責設計與使用者互動的介面或媒介，如網站、APP。主要工作內容為電腦系統的視覺呈現與使用體驗，除平面設計關注的介面美觀設計，也重視網頁功能性、操作性、便利性。舉凡區塊顏色、頁面排版、字型與字體大小、按鈕位置與設計、重點資訊凸顯，都是 UI 設計師重視的要點。 適合具有美感、善於處理資訊關係、具有網站設計經驗（多元互動、微動畫設計）、跨領域溝通能力佳的你。

Web/App PM

Product Manager

Web/App PM

是產品的領導人，同時為開發團隊中職位較高的要角，通常為團隊主管、組長等的領導職。負責掌握市場走向、擬定產品路線圖，除了技術面得具備一定程度知識，也需了解市場和行銷資訊，隨時確認產品發展在正確之道，掌控穩定獲利的商業模式。Product Manager 通常具市場敏銳度、UX 與跨領域知識、 3 -5 年以上工作經驗與對產品的極大熱忱。 適合能與各種人共事、快速切換工作項目、邏輯明確又細膩、遇到麻煩願意主動出擊的你。
