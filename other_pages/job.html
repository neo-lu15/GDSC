Product Designer

user interface & user experience

Product Designer

Product Designer 的工作內容多元繁雜，多為業界具有 3-5 年資歷的 UI 或 UX 設計師擔任或轉職而成，偶爾也會有少見的 Junior 職位。產品設計師需要多方了解關於產品的知識，才能更了解設計符合市場需求、獲利機會高產品所具備的要素。該職位能體驗到規劃產品願景、走向、功能等等前期作業的經驗，Junior 若感興趣可以多加把握。

UI/UX Designer

user interface & user experience

UI/UX Designer

綜合了「UX Designer」和「UI Designer」的職位！工作內容包含許多層面，從起初前期研究、產品架構、使用者易用性測試到最終產品的介面設計，都是 UI/UX Designer 可能接觸的範圍，實際工作內容視各公司需求與工作流程而有變動。

UI 相關職位

Web/App UI designer

user interface designer

Web/App UI designer

UI設計師負責設計與使用者互動的介面或媒介，如網站、APP。主要工作內容為電腦系統的視覺呈現與使用體驗，除平面設計關注的介面美觀設計，也重視網頁功能性、操作性、便利性。舉凡區塊顏色、頁面排版、字型與字體大小、按鈕位置與設計、重點資訊凸顯，都是 UI 設計師重視的要點。

Graphic Designer

user interface designer

Graphic Designer

又名 Art Designer。工作內容包含視覺設計、企業 CIS 識別設計、網站和 App 所需圖片素材、產品包材 / DM / 活動小卡設計優化、參與活動專案設計執行、品牌周邊商品設計發想等。製作美觀且符合需求的視覺設計作品、對字型設計的了解與選擇、色彩的調配、敏銳的美感和細節感知，都是 Graphic Designer 所注重的技能。

UX 相關職位

UX Researcher

user experience designer

UX Researcher

UX Designer 會根據 UX Researcher 的研究資料分析用戶的行為與想法，並提出可行解決方案執行。隨時「將使用者放第一」是他們的工作宗旨，會經手的的工作內容相當豐富。舉凡常聽到的產品架構圖、 介面流程規劃 User Flow、使用者旅程地圖 User Journey Map、框架圖 Wireframe、原型 Prototype 等，都是 UI Designer 可能負責的項目。

UX Designer

Product Management 相關職位

Project Manager

product management

Project Manager

Project Manager（專案經理）是負責監督專案進度者。通常是由該職位向客戶或老闆承接專案或新需求，接下來與設計師和工程師溝通，確認兩方完成案子所需的時間、擬定專案開發時程，並在其他專案時遇到的問題時給予協助，確保專案進行順利。

Web/App PM

product management

Web/App PM

是產品的領導人，同時為開發團隊中職位較高的要角，通常為團隊主管、組長等的領導職。負責掌握市場走向、擬定產品路線圖，除了技術面得具備一定程度知識，也需了解市場和行銷資訊，隨時確認產品發展在正確之道，掌控穩定獲利的商業模式。Product Manager 通常具市場敏銳度、UX 與跨領域知識、 3 -5 年以上工作經驗與對產品的極大熱忱。

Front-end

前端工程師

Front-End Engineer 負責撰寫前端程式，簡單來說，就是將設計師完稿的圖轉變成真正可以操作的數位產品，如：網站、App等等。前端設計師會依照設計稿完成畫面佈局配置，包括字型設定、按鈕大小等，是仰賴高度與設計師合作的職位！
