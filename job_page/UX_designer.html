User Experience UX Designer

職位任務

UX Designer 會根據 UX Researcher的研究資料分析用戶的行為與想法，並提出可行解決方案執行。隨時「將使用者放第一」是他們的工作宗旨，會經手的的工作內容相當豐富。舉凡常聽到的產品架構圖、 介面流程規劃 User Flow、使用者旅程地圖User Journey Map、框架圖 Wireframe、原型 Prototype 等，都是 UI Designer 可能負責的項目。

tip

適合觀察力好、思考邏輯佳、喜歡參與討論、總是對於解決問題有源源不絕的熱情和耐心的你。
